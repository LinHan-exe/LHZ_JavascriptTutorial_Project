Arrays
Arrays are a unique kind of variable that can store multiple values. Think of an array like a pokemon collection. You'll only have 1 collection but inside the collection there could be more than just 1 pokemon card. Arrays have values stored at specific index numbers, so you can easily retrieve the data using those indexes. You can also run a loop to fetch array data easier!
