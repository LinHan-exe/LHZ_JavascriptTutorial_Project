Conditionals
Conditional statements or conditionals control behavior in Javascript in order to determine whether or not a piece of code should run. There are 3 basic conditional statements: The "If" statement, the "Else" statement, and the "Else If" statement. For the If Statement, if the conditional is set to true, the code runs. For the else statement, if the same conditional is set to false, the code runs. The Else If statement runs a new conditional test if the first conditional is false.
